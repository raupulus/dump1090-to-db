# docs/info

Documentación técnica extendida del proyecto y registro de decisiones de arquitectura, en formato Markdown.

A diferencia de [AGENTS.md](../../AGENTS.md) (resumen operativo para agentes de IA) y del [README.md](../../README.md) del proyecto (guía de instalación y uso), aquí se documenta el *por qué* de las decisiones técnicas, la deuda técnica identificada y las notas de diseño que no encajan en los documentos anteriores.

## Índice

- [arquitectura.md](arquitectura.md) — flujo de datos y responsabilidad de cada componente.
- [decisiones.md](decisiones.md) — registro cronológico de decisiones técnicas y deuda técnica conocida.

## Convención

- Un fichero Markdown por tema (arquitectura, decisiones, integraciones externas, etc.); se pueden añadir nuevos ficheros según crezca el proyecto.
- Las entradas de `decisiones.md` se añaden al final en orden cronológico, sin reescribir el historial, indicando siempre la fecha.
